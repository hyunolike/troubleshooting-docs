## 문제
- `$'\r': command not found`

## 해결
- `DOS` `UNIX` 의 개행 방식 서로 달라서 생기는 문제
  - 윈도우의 개행 문자 방식 `CR LF` >> 여기서 CR(Carriage Return)으로 커서 제일 앞 이동! LF(Line Feed) 줄 바꿈!
  - 리눅스의 개행 문자 방식 `LF`
- `Notepad++` 프로그램을 통해 편집
